# MsgGo
![MsgGo](./app/src/main/res/drawable/icon.png)  


安卓平台上的一款轻量的短信群发App。  
## features
* 界面美观，交互性良好
* 基于excel数据格式导入
* 自动获取短信变量名
* 提供短信编辑器，支持短信**变量**代换
* 即时回馈信息发送状态，成功与否，尽在眼前
* 选择指定收件人无压力
* 无需指定固定格式，App内指定号码列
* *第三方应用数据一键导入*

使用步骤：  
1. 导入数据
2. 编辑短信内容
3. 选择号码变量
4. 选择收件人号码列
5. 发送

## Excel格式要求
列名|列名|列名|...
-|-|-|-
数据|数据|数据|...
数据|数据|数据|...
...|  
  
注：（未测试）兼容所有excel格式
